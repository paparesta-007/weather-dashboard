##  Installation

### 1. Clone the Repository
```bash
git clone https://github.com/paparesta-007/weather-dashboard
cd weather-dashboard
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development Server
```bash
npm run dev
```
